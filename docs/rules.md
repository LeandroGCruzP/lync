# Rules

### Organização
[  ] Qualquer usuário pode criar uma organização
[  ] Somente os ADMINs de uma organização podem editar a organização
[  ] Somente os ADMINs de uma organização podem convidar usuários para a organização
[  ] Somente os ADMINs de uma organização podem revogar o convite de um usuário para a organização
[  ] Somente os ADMINs de uma organização podem remover usuários da organização
[  ] Somente o OWNER de uma organização pode transferir a propriedade da organização para outro usuário
[  ] Somente o OWNER de uma organização pode deletar a organização

### Evento
[  ] Somente os ADMINs de uma organização podem criar eventos
